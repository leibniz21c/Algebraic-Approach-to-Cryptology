정답 코드는 수업날짜 3일후부터 열람하십시오.
그 전에는 절때 열람하시면 안됩니다.
실습언어는 편하신 언어를 사용해주시면 됩니다.

< 문제1 >
유클리드 호제법을 이용한 GCD를 구하는 프로그램을 구현합니다.

< 제공 >
본 문제에서는 제공사항이 없으며, 직접 프로그램이 제대로 동작했는지
확인부탁드립니다.

< 문제2 >
plain_text.txt 파일의 내용을 3가지 암호방식으로 부호화한다.
랜덤으로 뽑은 key 값을 사용하며, 자명한 암호 key는 제외한다.
각 방식별로 10개의 중복이 허용되는 key 값을 사용하는 암호문 파일을 생성하며
파일 형식은 다음과 같다.
code_add01.txt : 덧셈암호를 사용한 첫번째 코드파일
code_add02.txt : 덧셈암호를 사용한 두번째 코드파일
...
code_add10.txt : 덧셈암호를 사용한 열번째 코드파일

code_mult01.txt : 곱셈암호를 사용한 첫번째 코드파일
...
code_mult10.txt : 곱셈암호를 사용한 열번째 코드파일

code_affine01.txt : 아핀암호를 사용한 첫번째 코드파일
...
code_affine10.txt : 아핀암호를 사용한 열번째 코드파일

< 제공 >
강사는 무차별 공격 프로그램으로 암호를 해독할 것이며,
아핀 암호방식의 경우는 문자 빈도 분석 프로그램으로 추가 공격을 진행합니다.
이후 각 파일마다 해독에 걸린 연산수를 제공합니다.
